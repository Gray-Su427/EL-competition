# 开发指南-v3

## 项目概述
南京大学鼓楼校区校园餐饮推荐应用 "今天吃什么"。

三部分组成：
- **前端** — React 19 + TypeScript 6 + Vite 8 SPA（React Router 路由）
- **后端** — FastAPI + SQLAlchemy + SQLite
- **壳应用** — Android Kotlin Compose + WebView

### 命令
| 命令（`frontend/` 下执行） | 说明 |
|---|---|
| `npm run dev` | 启动 Vite 开发服务器 |
| `npm run build` | `tsc -b && vite build`，输出到 `dist/` |
| `npm run lint` | ESLint 检查 |
| `npm run preview` | 预览构建产物 |

### TypeScript 约束
| 选项 | 含义 |
|---|---|
| `erasableSyntaxOnly: true` | 禁止 enum、namespace、参数属性 |
| `verbatimModuleSyntax: true` | 必须显式写 `import type` |
| `noUnusedLocals` / `noUnusedParameters` | 未使用的变量/参数报错 |

### API 调用方式
- `config.ts` 导出 `API_BASE = 'http://localhost:8000'`
- `mock/mockApi.ts` 所有函数通过 `fetch` 调用后端 API
- `aiService.ts` 请求 `POST /api/ai/chat`，不包含硬编码 Key

### 初始 Mock 数据参考
7 道菜品、3 个食堂的种子数据，供后端参考。

## 后端 (`what_to_eat_today_web/backend/`)

### 技术栈
- FastAPI + SQLAlchemy + SQLite
- uvicorn 运行，httpx 代理 MiMo API
- python-dotenv 管理环境变量

### 目录结构（推荐）
```
backend/
├── main.py                ← FastAPI 入口，CORS 中间件，注册路由
├── database.py            ← SQLite 引擎 + Session 工厂
├── models.py              ← Canteen、Dish SQLAlchemy 表
├── schemas.py             ← Pydantic 响应模型
├── seed.py                ← 种子数据
├── routes/
│   ├── canteens.py        ← GET /api/canteens
│   ├── dishes.py          ← GET /api/dishes/recommended + /api/dishes/search
│   ├── suggestion.py      ← GET /api/suggestion/today
│   ├── search.py          ← GET /api/search/hot-keywords + /api/search/suggestions
│   └── ai.py              ← POST /api/ai/chat（httpx 代理 MiMo）
├── .env                   ← MiMo API Key
└── requirements.txt       ← fastapi, uvicorn[standard], sqlalchemy, python-dotenv, httpx
```

### API 端点
| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/canteens` | 获取食堂列表 |
| GET | `/api/dishes/recommended` | 获取推荐菜品列表 |
| GET | `/api/dishes/search?keyword=xxx` | 搜索菜品 |
| GET | `/api/suggestion/today` | 获取今日推荐文案 |
| GET | `/api/search/hot-keywords` | 获取热门搜索关键词 |
| GET | `/api/search/suggestions?keyword=xxx` | 搜索建议（输入联想） |
| POST | `/api/ai/chat` | AI 对话代理 |

### 种子数据
南京大学鼓楼校区三个食堂：
- 一食堂（川菜窗口、小吃窗口、麻辣烫窗口）
- 二食堂（面食窗口、家常菜窗口）
- 三食堂（快餐窗口、轻食窗口）

菜品数据参考 `frontend/src/mock/mockApi.ts` 中的 7 道 mock 菜品。

## Android (`What_to_eat_today_app/`)

### 修改内容

| 文件 | 改动 |
|---|---|
| `app/src/main/java/.../MainActivity.kt` | WebView `loadUrl("file:///android_asset/index.html")`，启用 `allowFileAccess` |
| `app/src/main/AndroidManifest.xml` | 添加 `android:usesCleartextTraffic="true"` |
| `app/src/main/assets/` | 放置前端构建产物 |

### 权限说明
- `INTERNET` 权限已有（用于调用后端 API）
- `usesCleartextTraffic="true"` — 允许 HTTP 明文流量（后端开发阶段使用 localhost）
- `allowFileAccess = true` — 允许加载 `file://` 协议下的本地 HTML

### 构建部署流程
1. `cd what_to_eat_today_web/frontend && npm run build`
2. 将 `frontend/dist/` 下所有文件复制到 `What_to_eat_today_app/app/src/main/assets/`
3. Android Studio 构建 APK

## 实施顺序
